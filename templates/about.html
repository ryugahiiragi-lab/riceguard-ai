Tentang RiceGuard AI

🌾 Latar Belakang Proyek

Sistem ini dibangun sebagai bagian dari proyek penelitian tugas akhir (Skripsi) untuk mendeteksi penyakit tanaman padi secara otomatis. Dengan memanfaatkan teknik visi komputer, sistem diharapkan mampu menjadi inovasi yang mempermudah identifikasi dini gangguan tanaman di sektor pertanian, khususnya pada studi kasus di Pertanian RW 02 Sumpiuh - Kabupaten Banyumas.

Detail Spesifikasi Teknis

Teknologi di balik arsitektur RiceGuard AI

🧠

EfficientNet-B0

Arsitektur Deep Learning

Model Convolutional Neural Network (CNN) yang menggunakan metode compound scaling untuk menyeimbangkan kedalaman, lebar, dan resolusi jaringan secara merata. Dipilih karena mampu menghasilkan tingkat akurasi ekstraksi fitur klasifikasi yang tinggi dengan beban komputasi yang efisien.

🔥

PyTorch & timm

Framework Kecerdasan Buatan

Pustaka machine learning sumber terbuka yang kuat untuk membangun tensor dan melatih model AI. Dipadukan dengan pustaka PyTorch Image Models (timm) guna mengimplementasikan struktur pre-trained dari arsitektur gambar mutakhir secara cepat dan stabil.

⚙️

Flask Python

Web Server Terintegrasi

Sebuah micro web framework berbasis Python yang ringan namun tangguh. Bertugas sebagai backend API yang menjembatani interaksi pengguna, mengeksekusi model AI secara real-time untuk prediksi gambar, dan mengelola logika riwayat ke dalam basis data MySQL.

🎨

Tailwind CSS

Gaya Antarmuka Komponen

Framework CSS dengan metode utility-first yang memfasilitasi perancangan antarmuka (UI) dengan sangat pesat tanpa hambatan, selaras dengan metode perancangan Rapid Application Development (RAD). Menghasilkan tampilan yang rapi, dinamis, dan responsif.
